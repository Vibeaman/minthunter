# MintHunter 🎯

NFT floor alerts + auto-mint Telegram bot for Ethereum.

## Features
- 🔔 Floor price alerts
- ⚡ FCFS competitive minting (multi-RPC, pre-signed tx)
- 👛 Wallet management (AES-256 encrypted keys)
- 🐋 Whale tracking
- 🔥 Trending collections

## Setup
```bash
cp .env.example .env
# Edit .env with your bot token and Alchemy RPC
npm install
npm start
```

## Fee
$0.50 (~0.0002 ETH) per FCFS mint to support development.
